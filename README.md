# 👆 Clicker Game

A simple clicker game built with javascript. This is my first project and the one that started it all!

## 📄 Tech Stack
- Javascript
- HTML
- CSS

## ✨ Features
Heres what you can do:
- **Click**: Duh.
- **Upgrade Your Clicks**: Once you meet the click requirement, you can upgrade you click per clicks.
- **Auto Clickers**: Add clickers to click for you.
- **Download/Upload Progress**: You can download or upload your progress.
